Getting Started

GDScript evolved as a custom scripting language to be used with the Godot computer (and mobile device) Game engine, and for those people who look beyond the real world, it also supports Virtual Reality.

Steps in getting started.

  1. Download Godot-Most recent version.
  2. Set up Visual Studio Code extension for Godot
  3. Confugure your enviroment preferences in the editor settings
  4. Start a project within Godot and open it again in Visual Studio Code
  5. Code up Hello World and enjoy.
Inputs to functions are called arguments. There may be no arguments, a list of arguments, type specified arguments, and arguments with default values. extends Node2D # Called when the node enters the scene tree for the first time. func _ready(): add(5, 6) # Prints 11 to Output window var sum = get_sum(2, 4) # Sets sum to 6 var my_int = add_ints(sum, 4) # Sets my_int to 10 my_int = times_2(my_int) # sets my_int to 20 move_x(self, my_int) # Move this node 20 pixels along x axis move_x(self) # Move by the default value # This function has no return value func add(a, b): print(a + b) # This function returns a value func get_sum(a, b): return a + b # This function will only accept integer arguments func add_ints(a: int, b: int): return a + b # Generate an error if the return value is not an int func times_2(n) -> int: return 2 * n # This function modifies an object that is passed by reference func move_x(node: Node2D, dx = 1.5): node.position.x += dx
